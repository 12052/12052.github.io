第3讲：三维空间刚体运动

0 综述

一个刚体在三维空间中的运动描述由一次旋转加一次平移组成。《视觉SLAM十四讲：从理论到实践》书中例题解析

1 直接上代码

代码使用的是Eigen库，Eigen适用范围广，支持包括固定大小、任意大小的所有矩阵操作，甚至是稀疏矩阵；支持所有标准的数值类型，并且可以扩展为自定义的数值类型；支持多种矩阵分解及其几何特征的求解；它不支持的模块生态系统，提供了许多专门的功能，如非线性优化，矩阵功能，多项式解算器，快速傅立叶变换等。

    1.2 旋转向量

    我们可以用一个向量，其方向和旋转轴一致，长度等于旋转角，成为旋转向量。根据罗德里斯格公式完成转换。

    1.3 欧拉角

    欧拉角提供了一个直观的方式来描述旋转——使用三个分离的转角，把一个旋转分解成3次绕不同轴的旋转，有XYZ、YXZ等旋转方式，如航空中描述的“偏航——俯仰——翻滚”(yaw-pitch-roll)三个角度描述一个旋转。相当于ZYX。

但是因为欧拉角的奇异性（万向锁问题），我们很少在SLAM程序中直接使用欧拉角表示姿态，同样不会在滤波或者优化中使用欧拉角表示旋转。

四元数

我们找不到不带奇异性的三位向量描述方式，三位旋转是一个三维流形，不可能无奇异的表达它，因此我们用四元数来表示旋转：

  • 首先，把三维空间点用一个虚四元数来描述p
  • 用四元数q表示这个旋转q
  • 则，p’即可表示这样的乘积。

我们高中的时候应该都学过复数，一个复数由实部和虚部组成，即x = a + bi，i是虚数单位，如果你还记得的话应该知道i^2 = -1。而四元数其实和我们学到的这种是类似的，不同的是，它的虚部包含了三个虚数单位，i、j、k，即一个四元数可以表示为x = a + bi + cj + dk，我们使用$q=((x,y,z),w) = (v,w)$，其中v是向量，w是实数，$q = ((x,y,z)sin(\theta /2),cos(\theta /2))$，如果我们想要把空间的一个点P绕着单位向量轴u = (x, y, z)表示的旋转轴旋转θ角度，我们首先把点P扩展到四元数空间，即四元数p = (P, 0)。那么，旋转后新的点对应的四元数（当然这个计算而得的四元数的实部为0，虚部系数就是新的坐标）为：

总结

旋转矩阵（3x3）：Eigen::Matrix3d;
旋转向量（3x1）：Eigen::AngleAxisd;
欧拉角（3x1）：Eigen::Vector3d。
四元数（4x1）：Eigen::Quaterniond。
欧式变换矩阵（4x4）：Eigen::Isometry3d。
仿射变换（4x4）：Eigen::Affine3d。
射影变换（4x4）：Eigen::Projective3d。
