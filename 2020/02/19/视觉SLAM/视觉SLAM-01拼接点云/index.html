  • 四元数
    只需要一个四维的四元数就可以绕任意过原点的向量的旋转，方便快捷

我们高中的时候应该都学过复数，一个复数由实部和虚部组成，即x = a + bi，i是虚数单位，如果你还记得的话应该知道i^2 = -1。而四元数其实和我们学到的这种是类似的，不同的是，它的虚部包含了三个虚数单位，i、j、k，即一个四元数可以表示为x = a + bi + cj + dk，我们使用$q=((x,y,z),w) = (v,w)$，其中v是向量，w是实数，$q = ((x,y,z)sin(\theta /2),cos(\theta /2))$，如果我们想要把空间的一个点P绕着单位向量轴u = (x, y, z)表示的旋转轴旋转θ角度，我们首先把点P扩展到四元数空间，即四元数p = (P, 0)。那么，旋转后新的点对应的四元数（当然这个计算而得的四元数的实部为0，虚部系数就是新的坐标）为：
$p’ = qpq^(-1)|$，其中，四元数乘法：$q1q2=(v_1\times v_2 + w_1v_2 + w_2v_1,w_1w_2-v_1*v_2)$
